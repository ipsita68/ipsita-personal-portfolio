Spam Detector Project

  • Description: Developed a web-based Spam Detector application using Python and Flask to classify messages as spam or ham (non-spam) based on machine learning algorithms. Integrated pandas for data manipulation, scikit-learn for machine learning tasks, and imbalanced-learn for handling imbalanced datasets. Engineered features using TF-IDF Vectorizer to convert text data into numerical features and implemented Multinomial Naive Bayes classifier for message classification.   • Technologies Used: Python, Flask, pandas, scikit-learn, imbalanced-learn
  • Functionality: Implemented data preprocessing techniques to prepare the text data for classification. Utilized machine learning algorithms to train a model capable of distinguishing between spam and non-spam messages. Integrated imbalanced-learn library to address class imbalance in the dataset, ensuring accurate predictions. Developed a user-friendly web interface using Flask for users to interact with the application. Collaborated with team members for efficient deployment and ongoing maintenance of the application.
  • Personal Contributions: Led the development of the Python scripts for data preprocessing, feature engineering, and model training. Implemented machine learning algorithms and tuned hyperparameters for optimal performance. Contributed to the integration of Flask for building the web interface and handled backend functionality. Ensured robust error handling and user feedback mechanisms for a seamless user experience.
